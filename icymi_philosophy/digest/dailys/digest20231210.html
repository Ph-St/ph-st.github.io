This was a nice outcome.

I'm trying to convert all my old papers to Quarto to make them more accessible, and more future-proof. And I thought it would be fun to use margin notes rather than footnotes where possible, because margin notes are more readable, and a better use of large paper.

Sometimes this was impossible because long notes don't work in the margins. In those cases I've stuck with footnotes.

But there was a technical problem. The way Quarto was written, it crashed outputting multi-paragraph notes in the margin.

So I learned how to file bug reports (something I should have done a while ago), and within weeks it was fixed. I just downloaded the latest pre-release build, and the margin notes are working.

This has definitely increased my credence that Quarto is a good tool to be using for writing.
