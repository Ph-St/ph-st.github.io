Thomas Hodgson

@twsh@scholar.social

Week 6 Lewis 'Scorekeeping in a language game'
https://doi.org/10.1007/BF00258436

Lewis' main idea is that the state of a conversation can include things like standards of precision. Does France count as hexagonal? If so, Italy counts as boot shaped.

Standards can be changed by accommodation. If someone says something that is true only if the standards are like so, they become so.

One focus of discussion was Lewis' claim that it is easy to raise standards, but hard to lower them.
