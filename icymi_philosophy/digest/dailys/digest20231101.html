Richard Zach

@rrrichardzach@mathstodon.xyz

@standefer What happens is this: Suppose you want to say that \(\forall\) x A(x) is only a formula if x is free in A(x). Then you have to define "free" simultaneously with "formula". For a beginning student, this is hard to understand. Requiring just "x occurs" instead of "x is free" doesn't work, because x occurs if A(x) contains a quantifier binding x. So your only option is to disallow that possibility: \(\forall\) x A(x) is a formula if x occurs in A(x) but neither \forall x nor \exists x do.

You now have two advantages: in a beginning logic book, you can mostly avoid substitution into formulas with free variables. You only need it for \forall E, i.e., going from \forall x A(x) to A(t). And here you can safely just replace all x by t because x can't occur bound in A(x). \exists I is also easier: in going from A(t) to \exists x A(x) by replacing one or more occurrences of t by x you're safe because x can't occur bound in A(t) as long as \exists x A(x) ends up being well-formed. You can only use x's that don't occur in A(t) at all.

The downside is that in advanced texts people usually do use the standard approach, and then students get confused/have to be retrained.
