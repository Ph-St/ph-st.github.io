What is attention?

This is a fraught question, but it actually has a simple answer whose virtue is that every scientist of attention accepts it. This answer identifies a common ground conception of attention (to use a linguistic notion), what is in the background when we talk to each other about attention. The problem is that it has stayed too far in the background.

Here‚Äôs the definition: a subject's attending to X is the subject's mentally selecting a target to respond to. Actually, the entailed sufficient condition is enough for the science of attention.

Why is this common ground? Because our experimental methods are built on it . If so, then we‚Äôre all committed to this account just by being committed to the standard experimental paradigms.

Studying attention requires us to set or direct an experimental subject‚Äôs attention. Everyone knows how to do this. A good task requires as a necessary condition of correct task performance that a subject mentally select (e.g. visually, cognitively) a target to respond in the way instructed. That is how we control attention so as to study a capacity that is uniformly implemented trial to trial (or we can measure this by measuring performance).

This is a methodological assumption, a regulative principle. It provides a basis for constructing new paradigms but is built into how we study attention. When task instructions set mental (e.g. visual) selection this way, they set attention. Our instructions just to apply the sufficient condition.
